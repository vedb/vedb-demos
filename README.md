# vedb-demos

These notebooks demonstrate the basics of loading and manipulating files in Python for data analysis.

**VEDB_demo_explore_session.ipynb** Explains the file structures, demonstrates loading various files, and visualizes some gaze calibration components.

**VEDB_example_face_hand.ipynb** Shows how gaze and world video streams can be used together to answer basic research questions: where are faces and hands in our visual diet, and how frequently do we fixate on faces and hands?
